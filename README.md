# LLM VRAM Calculator - Hugo Site

This is a standalone Hugo site containing the LLM VRAM Calculator.

## Development

1. Install Hugo (extended version recommended).
2. Run `hugo server` to start the development server.
3. Open `http://localhost:1313` in your browser.

## Deployment to GitHub Pages

1. Create a new repository on GitHub.
2. Push this directory to the repository.
3. Go to Repository Settings > Pages.
4. Set Build and deployment to **GitHub Actions**.
5. GitHub will automatically detect the Hugo setup and provide a workflow.

## Structure

- `layouts/index.html`: The main calculator page.
- `hugo.toml`: Site configuration.
